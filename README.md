# 🌉 CashBridge — فاتورتك مال، خُذه الآن

> منصة تمويل فوري للفواتير للمستقلين وأصحاب المشاريع الصغيرة في العالم العربي

[![Live Demo] https://cash-bridge-app.netlify.app/

---

## 🔴 المشكلة

بحسب إحصائيات 2025:

- **50 مليون+** مستقل وصاحب مشروع في العالم العربي
- **51%** من فواتير B2B تُدفع متأخرة — متوسط انتظار **60–90 يوماً**
- **95%** من البنوك ترفض تمويل المشاريع الصغيرة بدون ضمانات
- خسارة **8–10%** من قيمة كل فاتورة على رسوم التحويل
- فجوة تمويل تتجاوز **123 مليار دولار** في المنطقة

### قصة خالد

> خالد مقاول، اشتغل أسبوعين على تشطيب شقة بـ **3,000$**.
> سلّم الشغل — الشركة قالت: *"بعد 60 يوم."*
> راح عالبنك — رفضوه بدون ضمانات.
> استنى شهرين، واستلم **2,700$** بدل 3,000$.

---

## 💡 الحل — CashBridge

منصة تحوّل أي فاتورة معتمدة إلى **كاش فوري خلال 24 ساعة**
بدون بنك — بدون ضمانات — بضغطة زر واحدة.

```
فاتورة 3,000$ → تمويل فوري → 2,550$ في حسابك بكرا
```

---

## ⚡ كيف يعمل

```
1. أصدر الفاتورة     →  فاتورة احترافية ترسل مباشرة للعميل
2. العميل يوافق      →  تتحول لضمان رقمي ملزم قانونياً تلقائياً
3. AI يقيّم فوراً    →  بناءً على سجل عملك — لا بنك، لا ضمانات
4. الكاش في حسابك   →  85% خلال 24 ساعة، الباقي عند دفع العميل
```

## 💰 نموذج الإيرادات

| البند | التفاصيل |
|---|---|
| العمولة | 3% من قيمة كل فاتورة ممولة |
| مثال | فاتورة 3,000$ → عمولة **90$** |
| للمستخدم | يستلم 2,550$ فوراً + 360$ بعد دفع العميل |
| بدون | رسوم اشتراك · رسوم خفية · ضمانات |

## 🛠️ مجموعة التكنولوجيا

| الطبقة | التقنية |
|---|---|
| **Frontend** | React.js 18 + Custom CSS |
| **Architecture** | SPA — Single Page Application |
| **Fonts** | Cairo (عربي) + Space Mono |
| **Auth Flow** | JWT-based (Simulated in MVP) |
| **AI Scoring** | Invoice history-based credit model |
| **Payments** | Stripe API + Wise API (planned) |
| **Hosting** | Netlify |
| **Backend (planned)** | Node.js + Express + PostgreSQL |

## 🚀 تشغيل المشروع محلياً

```bash
# clone المشروع
git clone https://github.com/your-username/cashbridge.git

# ادخل للمجلد
cd cashbridge

# افتح الملف مباشرة في المتصفح
open index.html

# أو استخدم live server
npx live-server
```

> لا يحتاج npm install — يعمل مباشرة في المتصفح
